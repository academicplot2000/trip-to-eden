## ✦ About

**Trip to Eden** is an immersive, storytelling-driven web experience designed to simulate a journey towards finding one's soulmate. Through a series of interactive chapters—from receiving a mysterious letter to selecting a seat on a celestial flight—users are guided through a reflective process that culminates in a personalized questionnaire.

The project emphasizes aesthetic minimalism, smooth animations, and an ethereal atmosphere, inviting users to pause and engage with the narrative.

## ✦ Features

*   **Immersive Atmosphere**: Dynamic sky backgrounds with floating cloud animations.
*   **Interactive Storytelling**: 
    *   **The Invitation**: A digital envelope reveal with a slide-to-open mechanic.
    *   **The Flight**: A cabin seat selection interface where every choice has a poetic meaning.
*   **Questionnaire Engine**: A 12-step form featuring sliders, multiple-choice, and open-text inputs.
*   **Internationalization**: seamless English (EN) and Spanish (ES) language toggling.
*   **Responsive Design**: Optimized for a fluid experience across devices.

## ✦ Structure

The project is built with vanilla web technologies, ensuring lightweight performance and easy maintenance.

```plaintext
Trip-to-eden/
├── index.html          # Main application structure and layout
├── styles.css          # Visual styling, animations, and responsive rules
├── script.js           # Core logic, event handling, and UI interactions
├── translations.js     # Internationalization dictionaries (EN/ES)
├── logotriptoeden.png  # Project branding asset
└── carta.png           # Letter/Envelope asset
```

## ✦ Getting Started

This project requires no build tools or package managers. It is ready to run directly in any modern web browser.

1.  **Clone or Download** the repository.
2.  **Open** the `index.html` file in your preferred browser (Chrome, Firefox, Safari, Edge).
3.  **Enjoy** the journey.
